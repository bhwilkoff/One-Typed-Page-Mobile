About One Typed Page

This mobile viewer is built for One Typed Page, a daily project by Daniel Marleau.

One Typed Page is a commitment to type one page every day on a manual typewriter, sharing the raw, unedited results. It's about the process, the practice, and the physical act of typing.

About the project:
"Each page is typed once, with no corrections, no edits, no spell-check. What you see is what came out of the typewriter, mistakes and all. It's about embracing imperfection and finding beauty in the process."

Mobile viewer created as an independent project to enhance the One Typed Page experience.
